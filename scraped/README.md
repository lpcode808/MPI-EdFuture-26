# Public source snapshots

This folder holds the public source evidence used to build the guide:

- `day-1-program.png` — supplied public Day 1 program image
- `day-2-program.png` — supplied public Day 2 program image
- `humanitix-public-facts.json` — a sanitized capture of attendee-facing facts from the public Humanitix event listing on July 16, 2026

The full Humanitix page source is intentionally **not** stored. It contains checkout form markup, transient platform tokens, internal host identifiers, and unrelated Humanitix code that the attendee guide does not need. The sanitized capture preserves the relevant public facts, exact source URL, capture time, brand colors, and visible contradictions without widening the repository's data footprint.

The program images are the canonical source for session titles, published start times, rooms, and the few printed presenter names. They do not publish session end times or descriptions, so the app does not invent them.
